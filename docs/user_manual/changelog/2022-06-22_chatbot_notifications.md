# Chatbot notifications

Groups can now set up a chatbot integration that posts thread and poll updates to external chat platforms.

## What gets posted

When configured, the chatbot posts notifications to a chat channel when:
- A new thread is started
- A new poll is created
- A poll outcome is shared
- Voting is about to close (with a list of who hasn't voted yet)

The poll outcome post includes the winning option, making it easy to share decisions with people who use chat but not Loomio.

## Setup

Instance admins configure chatbot webhooks under Admin > Integrations. Group coordinators then enable the feature and set the destination channel from group settings.
