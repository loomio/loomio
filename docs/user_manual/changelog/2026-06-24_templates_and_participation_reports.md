# Templates and participation reports

Templates and reporting both received substantial updates.

## User-created templates in browse

User-generated discussion and poll templates can now appear in template browsing.

This makes it easier to discover useful templates created inside your organization, not only built-in examples.

## Forking templates

Templates now have a fork action.

Forking lets you copy a template and use it as a starting point for your own group or process. This is useful when an example is close to what you need, but you want to adapt the wording, settings, or poll structure.

## Template draft cleanup

When a discussion or poll template is saved, Loomio clears the collaborative editor draft for that form. This prevents old draft content from reappearing after the template has already been saved.

## Participation report improvements

The participation report now includes more ways to understand activity across groups and tags.

New reporting views include:

- group scope controls
- tag-based thread reporting
- tags per interval
- tagged threads by user
- chart improvements
- a user filter

Group scope controls were redesigned around:

- All groups
- My groups
- Custom

This makes it easier to run reports for the whole organization, only groups you belong to, or a chosen subset of groups.

## Tags in reports

Reports can now show how tags are being used over time and which users are tagging threads.

This is useful for tracking themes, work streams, campaigns, or recurring topics across an organization.
