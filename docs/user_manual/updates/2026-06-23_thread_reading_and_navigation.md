# Thread reading and navigation

Thread reading behavior and navigation were adjusted to better match what people expect after they participate.

## New to you

The thread navigation now includes New to you.

This gives people a clearer place to find activity they have not yet seen, without mixing it up with every recent update.

## Your own activity is marked read

When you add content to a thread, Loomio now marks your own new thread items as read for you.

This avoids the confusing pattern where your own comment, poll, vote, or outcome immediately appears as something you still need to read.

## Latest visibility fix

The Latest thread navigation was adjusted so it more accurately reflects visible thread activity.

## Participant and notification history

Participant history and notification history actions were restricted to the people who should be able to see them.

This keeps thread administration information from being exposed too broadly.
