# Smaller improvements (November)

- **Thread-start button disabled at limit.** When a group has reached its maximum number of open threads, the "Start thread" button is now disabled rather than letting you write a post that can't be saved.
- **Dashboard loads fewer items initially.** The dashboard now shows a smaller set of recent threads and polls by default for faster load times, especially on instances with many groups.
- **Email replies to notifications.** Replying directly to a notification email (rather than a thread email) now bounces back with a clear message rather than being silently discarded.
- **Group data exports include comment titles.** CSV and HTML exports of group data now include comment titles alongside body text.
