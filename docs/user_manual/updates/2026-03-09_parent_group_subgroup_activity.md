# Parent groups show subgroup activity

Viewing a parent group's Discussions or Polls panel now shows threads from its subgroups as well as the parent group itself, giving admins and members a fuller picture of organization-wide activity without switching between subgroups.

Separately, people with permission to add guests to a group can now find and invite users who belong to a different organization — previously the invite search silently excluded anyone outside the inviter's own organization.
