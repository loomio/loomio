# Smaller improvements (2023)

- **Thread templates.** Coordinators can now create reusable thread templates with default text and settings. Start a discussion from a template to standardize recurring processes like meeting agendas or project updates.
- **Canonical URLs.** Thread and group pages now include a canonical URL tag, helping search engines understand which URL is the primary one and improving search result quality.
- **Link previews.** Pasting a URL into the editor now generates a preview card showing the page title, description, and image. This works for most websites and makes shared links more readable in threads.
- **Bot accounts.** Instance admins can mark user accounts as bots. Bot accounts are excluded from member counts and don't appear in member lists, but can still post and interact via the API.
- **Custom login provider names.** Instance admins can now set custom display names for OAuth and SAML login buttons (e.g., "Sign in with Organization Account" instead of the provider's default name).
- **Notification email sender.** Notification emails now show the site name rather than "Anonymous User" in the from field.
- **Editor upgrade.** The rich text editor was upgraded with better task list support and more reliable formatting.
