# Smaller improvements (June)

- **Template galleries and poll type renames.** Poll templates are now organized into a gallery, making it easier to find the right poll type. Several poll types have clearer names.
- **Sidebar remembers your preference.** The sidebar now stays open or closed based on your last choice, persisted across page loads.
- **Dark mode fixes.** Dark mode is now consistently applied in all areas of the app, including the thread page and editor.
- **Upload progress bars restored.** File uploads now show progress again.
