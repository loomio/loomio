# New poll type: Single Transferable Vote (STV)

Loomio added STV as a poll type for multi-winner, proportional elections, supporting Scottish (WIGM) and Meek counting methods with Droop or Hare quotas.

Voters rank candidates via drag-and-drop. Results show round-by-round counting, with ties flagged rather than silently broken. STV polls are always anonymous, and results stay hidden until the poll closes. Results can be exported in BLT format.

This is a beta feature: because the counting logic is new, the poll form displays a warning inviting people to report bugs on GitHub. Expect ongoing refinement.
