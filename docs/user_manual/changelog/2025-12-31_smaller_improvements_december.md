# Smaller improvements (December)

- **Typography and readability.** Headings in user-generated content are now bold, making threads easier to scan. General typography on thread pages has been tightened for better readability.
- **Dashboard performance.** The dashboard loads more quickly, especially for users belonging to many groups.
- **Poll option reliability.** Fixed a bug where deleting a poll option during poll creation could leave behind invalid data that caused voting errors.
- **Thread warning rewritten.** The warning shown when you try to start a discussion in a group that already has an active one has been reworded to be clearer.
- **Export includes author names.** CSV exports of group data now include the author name for each comment alongside the comment body.
