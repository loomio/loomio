# Smaller improvements (March)

- **Thread settings: allow comments, allow reactions.** Threads gained two new toggles in Thread settings. Turning either off stops participants from posting comments or adding emoji reactions on that thread, giving thread owners finer control over how a conversation can be used, for example an announcement-only thread.
- **Cohousing signup category.** When creating a new group, "Cohousing" is now available as a category option alongside Community, Government, and others.
