# Poll quorum

Polls can now require a minimum number of voters before a result is considered valid.

## How it works

When creating a poll, coordinators can set a quorum — the minimum number of people who must vote. Until the quorum is reached, the poll banner shows how many more votes are needed and the results panel notes that quorum hasn't been met. Once enough people have voted, the quorum message disappears and results are shown normally.

This is useful for boards and committees where a decision isn't binding unless a minimum number of members participate.

The quorum setting is available on all poll types and can be set in the poll creation form under "More settings."
