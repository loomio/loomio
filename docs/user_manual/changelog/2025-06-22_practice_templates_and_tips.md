# Practice templates and getting started tips

New groups now get a set of guided steps to help them start using Loomio effectively.

## Practice thread

Each new group starts with a practice thread and a practice proposal. These appear in the group with example content, giving members a safe place to experiment with discussions, polls, and voting before doing it for real.

## Getting started checklist

The sidebar now has a "Getting started checklist" that walks group admins through setup: inviting members, starting a first thread, running a poll, and sharing an outcome. Each step shows a brief tip on hover. Tips are shown for the first 28 days of a group's life and hide automatically when all steps are complete.
