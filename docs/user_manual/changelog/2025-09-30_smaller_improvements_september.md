# Smaller improvements (September)

- **@mentions on new discussions work reliably.** Fixed a bug where @mentioning someone in the first comment of a new thread would sometimes fail to send them a notification.
- **Email replies from unknown senders.** Replying to a notification email from an unrecognized address is now handled gracefully rather than silently dropped.
- **Dutch translations corrected.** Fixed an erroneous Dutch translation that swapped the meaning of two labels.
