# Interface refresh

Loomio’s interface has been refreshed as part of the move to a newer version of the design framework.

## What changed

The update touches many parts of the app, including:

- forms
- menus
- buttons
- dialogs
- poll voting screens
- thread event rows
- group settings
- member lists
- email settings
- report controls

The goal is a cleaner and more consistent interface across the app.

## Tags and thread controls

Tag controls were also adjusted during this work. Tag colours display more reliably, and newly added tags are checked immediately after they are saved.

Thread and poll action controls were tightened so actions are easier to scan and use consistently.

## Account merge screens

The account merge confirmation screens were updated along with the interface refresh, so the flow is clearer when combining duplicate accounts.

## What stayed the same

This was mostly an interface and consistency update. The underlying concepts of groups, threads, polls, votes, and memberships remain the same.
