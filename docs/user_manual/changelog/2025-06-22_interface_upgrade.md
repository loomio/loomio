# Interface refresh

A ground-up rebuild brings a faster, more responsive experience throughout the app.

## What changed

Page transitions between the dashboard, group pages, and threads are now instant. Thread pages with hundreds of comments load and scroll smoothly. The entire interface re-renders from a shared state, so navigating between threads in a group no longer reloads common data.

## Visual changes

Buttons, form fields, modals, and layout spacing are more consistent across the app. The sidebar defaults to open on large screens and remembers your open or closed preference across visits. Unread thread previews now have a prominent highlight that makes them easy to pick out in a long list.

## Editor

The rich text editor supports more formatting options and handles file attachments more reliably. Formatting buttons light up to show which styles are active on the current selection.
