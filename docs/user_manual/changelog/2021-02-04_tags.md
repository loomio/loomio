# Tags

Groups can now organize threads with tags.

## Managing tags

Group coordinators manage the group's tag list from the group settings page. Add, rename, reorder, or delete tags at any time. Tags can be assigned colors to make them easy to spot.

## Using tags

When creating or editing a thread, select one or more tags from the group's tag list. Tags appear on the thread card on the dashboard and on the group's discussions page, making it easy to filter and scan threads by topic.
