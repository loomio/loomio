# Thread layout settings and Files panel scope

The thread settings modal now offers two straightforward layout choices, Timeline or Threaded, replacing the previous combination of oldest/newest sorting with three separate nesting-depth options. A new "Mark as read" action has also been added to the thread's action menu.

A group's Files panel previously could surface attachments from certain visible subgroups as well. It now only shows files that belong to the group you're actually viewing.
