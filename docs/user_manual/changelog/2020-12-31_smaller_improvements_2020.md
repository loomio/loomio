# Smaller improvements (2020)

The second half of 2020 brought a collection of smaller fixes and refinements:

- **Email notifications improved.** The daily summary email now includes discussion and poll edits, not just new content. Notification emails were reorganized for consistency.
- **oEmbed replaces iframes.** Links to YouTube, Vimeo, and other supported services now embed inline as responsive previews rather than raw iframes.
- **Polish language added.** The interface is now available in Polish.
- **Load more counters.** The "Load more" button on threads, group discussions, and member lists now shows how many items remain (e.g., "Load 12 more").
- **Audience tweaks.** Controls over who can see and participate in threads and polls were refined based on user feedback.
