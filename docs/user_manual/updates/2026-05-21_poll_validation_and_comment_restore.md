# Poll condition indicators and comment restore permissions

When a poll has quorum or threshold conditions, each one now shows a clear pass (green checkmark) or fail (grey, turning red once the poll closes) indicator, both in the app and in emails and chatbot summaries.

Previously, if an admin discarded a member's comment, that same member could undo it themselves. Now, only the person who discarded a comment, or a group admin, can restore it. An author can still restore their own self-discarded comment, but not one an admin removed.
