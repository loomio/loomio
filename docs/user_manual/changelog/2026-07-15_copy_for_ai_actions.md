# Copy for AI actions

The **Copy for AI** dialog now offers three actions: copy the facilitation prompt and thread together, copy only the prompt, or copy only the thread. The prompt links to Loomio's facilitator skill, which includes guidance for analysing discussions, understanding concerns, and suggesting next steps.

The thread is copied as structured Markdown with its context, timestamps, replies, polls, visible results, vote reasons, and outcomes clearly identified. Results and vote reasons that are not yet visible to the member are not included, and voters remain unidentified in anonymous polls.

The actions are available to thread members. Permissions are unchanged.
