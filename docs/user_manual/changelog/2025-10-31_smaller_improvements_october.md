# Smaller improvements (October)

- **Dashboard "no polls" message.** When you have no polls waiting for your vote, the dashboard now shows a clear message instead of a blank panel.
- **Ranked choice poll clarification.** The "number of choices" field on ranked choice polls now has clearer helper text explaining what it does.
- **Error display when voting fails.** If your vote fails to save (for example due to a connection issue), the interface now shows a visible error rather than silently failing. An option to reload is offered when a page asset fails to load after a deploy.
