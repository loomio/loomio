# Template galleries and poll type renames

New "browse" pages let you explore the full library of poll and discussion templates, including a subgroup's parent-group templates, with a one-click "fork" action to copy an example into your own group.

## Poll type renames

Poll types were renamed for clarity: Choose, Allocate, Score, Rank, Time. A new Majority proposal template was added, and process introductions and subtitles were rewritten across templates.

## Simplified chooser

The "Show all templates in one list" option and the separate Meeting tab were removed in favor of a simpler template chooser.
