# Smaller improvements (August)

- **Mobile sidebar improved.** The sidebar on small screens now opens as an overlay rather than pushing content to the side, making thread reading on phones much smoother.
- **Reaction display on busy threads.** When dozens of people react, the reaction row now wraps cleanly and shows counts that are easier to read.
- **Group page thread list no longer overflows on mobile.** Panels with many pages of threads fit the screen width correctly on phones.
- **Invite button icon.** The "invite people" button now uses a bullhorn icon, making it faster to spot.
- **Print layout.** Page layout when printing a thread has been improved.
