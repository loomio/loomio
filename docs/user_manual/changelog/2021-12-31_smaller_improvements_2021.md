# Smaller improvements (2021)

- **Thread templates.** Coordinators can create reusable thread templates with default text, tags, and poll settings. Start a new thread from a template to save time on recurring processes.
- **Client reload on deploy.** When a new version of the app is deployed, the page now prompts you to reload instead of silently breaking, preventing confusion when features change.
- **Broadcast to guests.** When you add guests to a poll, they now receive a notification email. Previously only group members received poll notifications.
- **Links and tags in emails.** Notification emails now include thread tags and properly styled links, making it easier to scan emails and jump to the right thread.
- **Pinned threads.** Threads can now be pinned to the top of a group's discussion list. Pinned threads show a pin icon and stay at the top regardless of recent activity.
