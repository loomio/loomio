# Language and translation updates

Several translation and tone updates were made across Loomio.

## More consistent informal language

The app language was reviewed for consistency in several locales, including:

- German
- Spanish
- French
- Italian
- Brazilian Portuguese

The goal was to keep Loomio’s tone consistent and natural, especially in languages where formal and informal address can easily drift.

## Translation fixes for reports and tags

New reporting and tag-related strings were translated across supported client locales.

Several interpolation issues were corrected so translated strings keep the right variable names and render properly.

## Better translated notices

Server flash notices are now translated before being sent to the client.

This gives people clearer localized feedback in more situations.

## Why this matters

For multilingual groups, these updates make the interface feel more consistent and reduce confusing wording caused by automatic translation choices.
