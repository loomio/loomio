# Newsletter service

Groups can now send newsletter-style updates to their members via email.

From the group page, coordinators can compose and send a one-off email to all group members. The email includes rich text formatting, images, and links. Unlike discussion notifications, newsletters are a single broadcast — replies don't create a thread. This is intended for announcements, updates, and other communications that don't need a discussion.

Instance admins can enable or disable the newsletter feature at the instance level.
