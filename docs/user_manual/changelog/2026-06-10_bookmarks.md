# Bookmarks

Bookmarks give you a personal way to save important Loomio content so you can return to it later.

## What can be bookmarked

You can now bookmark several kinds of content:

- threads
- comments
- polls
- votes
- outcomes

Bookmarked content appears in a dedicated Bookmarks page, available from the sidebar.

## How bookmarks behave

Bookmarks are personal. Saving or removing a bookmark affects only your own account.

Removing a bookmark updates the bookmark count immediately and treats the item as no longer bookmarked. If you later save it again, it behaves like a fresh bookmark.

Logged-out visitors cannot create bookmarks, and logged-out paths now handle bookmark controls without raising errors.

## Why this matters

Bookmarks are useful for:

- keeping track of threads you need to return to
- saving a specific comment or vote for later reference
- collecting outcomes that matter to your work
- building a personal follow-up list without changing the thread for everyone else
